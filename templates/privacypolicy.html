Privacy Policy

Personal Data

SC2dashboard will not collect personal data when you visit the site. Your IP address will be stored in a short-lived server side log and will not be used for any purposes.

Battle.net API

All Data presented in sc2dashboard came from the Battle.net API only. In accordance with Blizzard's API policy, SC2dashboard provides data "AS IS" and "AS AVAILABLE". You shall expect no quality/performance/accuracy from the data provided by sc2dashboard
